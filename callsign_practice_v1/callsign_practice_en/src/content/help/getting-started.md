# Callsign Practice Guide

Use this page as the lightweight reference for the trainer.

## Quick Start

1. Press **Play Callsign**.
2. Listen to the transmission.
3. Type the answer or choose an option.
4. Continue through a 10-round session.

## Callsign Modes

### Transport + GA

Randomly alternates between airline callsigns and general aviation callsigns.

### Transport Airline

Format: **IATA code + 3 to 4 digit flight number**

Examples:

- `CA123`
- `MU2486`

You can answer either:

- the full callsign, such as `CA123`
- just the number, such as `123`

### General Aviation

Format: **aircraft type + suffix**

Examples:

- `Cessna DXF`
- `Piper A7K`

Answer the suffix only.

### Numeric Only

Standalone number groups with aviation-style pronunciation.

### Custom Letters

Uses your custom letter pool and configured length.

## Voice

The **Voice** panel controls:

- accent preset
- speaking rate
- optional radio-tone post-processing

## Other

The **More** entry opens secondary preferences such as:

- UI sound feedback
- vibration feedback

## Answer Modes

### Spelling

Type the answer and press **Enter** or **Check**.

### Multiple Choice

Choose the closest option, then press **Check** to confirm it.

## Replay And Reveal

- **Play Again** replays the current sample
- **Show Answer** reveals the current callsign and pronunciation

## Custom Letters

The **Custom Letters** shortcut appears only when the active mode is **Custom Letters**.
